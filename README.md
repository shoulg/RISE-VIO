# RISE-VIO

`rb_vins` was the original internal project name.

This repository contains the research code for **RISE-VIO**. The current implementation was developed primarily on top of **IC-GVINS** and builds on ideas from **drt_vio_init** and related prior works. The released version corresponds to the main implementation used in the paper. As with many research prototypes, the code is still being cleaned and documented, and some parts remain under active refinement.
